## Our Core Tools

- **Jolt** — All opening, closing, and daily checklists live here. Managers monitor compliance from the Jolt dashboard. Complete checklists on the store tablet; edits to templates are done via the Jolt website.
- **Zenput** — Temperature sensors for walk-in freezer and cooler. Real-time monitoring lets leadership see sensor readings across all locations remotely.
- **POS** — Order entry, sales reporting, and end-of-day cash reconciliation.

## Guiding Principles

1. **Consistency** — Every location runs the same way. Checklists exist so we don't rely on memory.
2. **Accountability** — Jolt tracks checklist completion. Missed tasks are flagged — don't skip, document.
3. **Communication** — Shift handoffs should include Jolt notes plus a verbal update on any open issues.

<Warning>
  All operational incidents (equipment failure, food safety issues, accidents) must be logged in Jolt the same day they occur. If Jolt is unavailable, notify the manager by text and document as soon as access is restored.
</Warning>
