# Daily Checklists

Use these checklists every shift. Physical sign-off sheets are kept at the manager station.

## Front of House — All Shifts

| Task | AM | PM | Close |
|------|----|----|-------|
| Check and restock host stand | ✓ | ✓ | |
| Wipe down all menus | ✓ | ✓ | ✓ |
| Inspect restrooms every 30 min | ✓ | ✓ | ✓ |
| Refill condiments and table items | ✓ | ✓ | |
| Bus and reset tables within 3 min of guest departure | ✓ | ✓ | |
| Communicate 86'd items to team | ✓ | ✓ | |

## Back of House — All Shifts

| Task | AM | PM | Close |
|------|----|----|-------|
| Verify cooler/freezer temps and log | ✓ | ✓ | ✓ |
| Label and date all prepped items | ✓ | ✓ | |
| Complete prep list before service | ✓ | ✓ | |
| Clean prep surfaces between tasks | ✓ | ✓ | ✓ |
| Take out trash when 3/4 full | ✓ | ✓ | ✓ |
| Complete deep clean of one station (rotating) | | | ✓ |

## Manager — Daily

- [ ] Review prior day's sales and flag variances
- [ ] Confirm staffing is complete for all shifts
- [ ] Check and approve inventory deliveries
- [ ] Address any open maintenance tickets
- [ ] Update the shift communication log
- [ ] Review and respond to any guest feedback

<Info>
  Completed checklists should be placed in the shift binder at the manager station. Do not discard — they are reviewed weekly.
</Info>
