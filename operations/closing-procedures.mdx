# Closing Procedures

The closing manager is responsible for completing the **Jolt Closing Checklist** before leaving the building. Nothing is done "from memory" — work through Jolt step by step.

<Info>
  Closing checklists in Jolt have a scheduled due time. If you won't finish by that time, notify the manager on duty. Missed closing checklists are flagged automatically.
</Info>

## Service Wrap-Up

<Steps>
  <Step title="Last guest out">
    Once the last guest departs, lock the entrance. Thank anyone still in the dining room and let them finish — do not rush guests.
  </Step>
  <Step title="Close the POS">
    Run the end-of-day POS report. Verify all transactions are settled. Document any voids, comps, or discrepancies with an explanation in the closing notes.
  </Step>
  <Step title="Cash handling">
    Count down the drawer and place the cash drop in the safe using the standard double-lock procedure. Do not leave cash on the counter.
  </Step>
</Steps>

## Closing Temperature Log

Complete the **Jolt Closing Temperature Log** before equipment shutdown.

| Item | Method | Required Range |
|------|--------|----------------|
| Walk-in Freezer | Zenput sensor | 0°F or below |
| Walk-in Cooler | Zenput sensor | 34–40°F |
| Sanitizer concentration | Test strip | Per chemical spec |

If sensor readings are outside range at close, **do not power down refrigeration**. Document the issue in Jolt and text/call Chad or the on-call manager.

## Kitchen Cleaning & Shutdown

- [ ] Drain and clean fryers; refill as needed for next day
- [ ] Scrape and clean all grill surfaces
- [ ] Clean custard machines per the end-of-night procedure
- [ ] Break down and sanitize all prep surfaces
- [ ] Clean and store all smallwares — nothing left soaking overnight
- [ ] Sweep and mop all kitchen and BOH floors
- [ ] Empty all trash cans and replace liners; take trash to the dumpster
- [ ] Run and empty the final dishwasher cycle

## Dining Room & Drive-Thru

- [ ] Wipe down all tables, chairs, and high chairs
- [ ] Clean condiment stations and restock for morning
- [ ] Clean and sanitize restrooms — leave fully stocked
- [ ] Sweep and mop dining room floor
- [ ] Wipe down drive-thru window, menu boards, and speaker
- [ ] Return headsets to charging station

## Equipment Shutdown

- [ ] Power down grills, fryers, and steam tables
- [ ] Verify custard machines are in overnight mode (per brand SOP)
- [ ] Turn off all non-essential lighting
- [ ] Set thermostat to overnight setting

## Lockup & Security

- [ ] Check all rear doors and side exits are locked
- [ ] Confirm walk-in freezer and cooler doors are fully closed and latched
- [ ] Submit the completed Jolt Closing Checklist
- [ ] Set the building alarm using your personal code
- [ ] Lock the main entrance — final manager out, no exceptions

<Warning>
  Never leave the property until the Jolt Closing Checklist is 100% submitted. If any item couldn't be completed, note why in Jolt before leaving — do not just skip it.
</Warning>
