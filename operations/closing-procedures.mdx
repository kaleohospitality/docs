# Closing Procedures

Follow all steps before leaving the property. The closing manager is responsible for signing off on the closing log.

## Last Guest & Service Wrap-Up

<Steps>
  <Step title="Clear the floor">
    Ensure all guests have departed. Thank departing guests and invite them back.
  </Step>
  <Step title="Close out all checks">
    Verify no open tickets remain in the POS system. Reconcile any discrepancies with the manager.
  </Step>
  <Step title="Complete tip-out and cash handling">
    Follow cash-handling procedures per the finance SOP. Place cash drops in the safe.
  </Step>
</Steps>

## Cleaning & Reset

- [ ] Break down and clean all tables and seating areas
- [ ] Clean and sanitize all surfaces (FOH and BOH)
- [ ] Sweep and mop all floors
- [ ] Clean restrooms and restock supplies
- [ ] Empty all trash and replace liners
- [ ] Run final dishwasher cycle and leave equipment clean and dry

## Equipment & Utilities

- [ ] Power down all non-essential equipment
- [ ] Set refrigerators/freezers and verify temperatures are logged
- [ ] Turn off all lighting except security lights
- [ ] Adjust thermostats to overnight settings

## Security & Lockup

- [ ] Confirm all back-of-house doors and windows are locked
- [ ] Set the alarm using your personal code
- [ ] Lock the main entrance and complete the closing log

<Warning>
  Never leave the property without completing the security checklist. If anything feels off, call the manager on duty before leaving.
</Warning>
